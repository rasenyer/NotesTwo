# NotesTwo
Simple aplicación Android de notas, desarrollada utilizando la arquitectura MVVM.

<h3>Características:</h3>
• Guardar nota
<br>
• Editar nota
<br>
• Buscar nota
<br>
• Eliminar nota

<h3>Librerías usadas:</h3>
• Room
<br>
• Viewmodel
<br>
• Livedata
<br>
• Coroutines
<br>
• Navigation Component
